基础表达式 ​

变量 ​

var ​

TIP

类型一旦确定后则不能再更改其类型

dart
var name = 'Bob';

Object ​

TIP

Object 是 Dart 所有对象的根基类,可以赋值任意对象

dart
Object name = 'Bob';

dynamic ​

TIP

dynamic和Object基本一样，不同的是dynamic声明的对象编译器会提供所有可能的组合，而Object声明的对象只能使用 Object 的属性与方法, 否则编译器会报错

需要格外注意，这很容易引入一个运行时错误,比如：字符串，没有"xx"属性，编译时不会报错，运行时会报错

dart
dynamic name = 'Bob';

可空类型 ? ​

TIP

定义为可空类型，对于可空变量，在使用前必须判空

dart
dynamic? name;

late ​

TIP

如果预期变量不能为空，但在定义时不能确定其初始值，则可以加上late关键字

表示会稍后初始化，但是在正式使用它之前必须得保证初始化过了，否则会报错

当一个 late 修饰的变量在声明时就指定了初始化方法，那么内容会在第一次使用变量时运行初始化,如果没有使用，则方法不会调用

dart
late dynamic name = 'Bob';

函数可选链 ​

TIP

fun 不为空时则会被调用

dart
fun?.call()

const ​

TIP

编译时常量（编译时直接替换为常量值）

dart
const name;

final ​

TIP

在第一次使用时被初始化为常量值

被final或者const修饰的变量，变量类型可以省略

虽然 final 对象不能被修改，但它的字段可能可以被更改。相比之下，const 对象及其字段不能被更改：它们是 不可变的

dart
final name;

操作符 ​

运算符参考https://dart.cn/language/operators

as is is! ​

TIP

类型转换（也用于指定库前缀）

is

如果对象具有指定类型则为 True

is!

如果对象不属于指定类型则为 True

?? ​

TIP

b是null，取value的值

dart
b ??= value;

TIP

aa是null，取value的值

dart
b =aa ?? value;

三元（和js一样 ​

TIP

condition ? expr1 : expr2

级联（..，?..） ​

TIP

对同一对象进行一系列操作

dart
var paint = Paint()
  ..color = Colors.black
  ..strokeCap = StrokeCap.round
  ..strokeWidth = 5.0;

等效于
var paint = Paint();
paint.color = Colors.black;
paint.strokeCap = StrokeCap.round;
paint.strokeWidth = 5.0;

TIP

如果级联操作的对象可以为空，则对第一个操作使用空短级联 ( )。以 开头可确保不会对该空对象尝试任何级联操作。

dart
querySelector('#confirm') // Get an object.
  ?..text = 'Confirm' // Use its members.
  ..classes.add('important')
  ..onClick.listen((e) => window.alert('Confirmed!'))
  ..scrollIntoView();

等效于
var button = querySelector('#confirm');
button?.text = 'Confirm';
button?.classes.add('important');
button?.onClick.listen((e) => window.alert('Confirmed!'));
button?.scrollIntoView();

dart
[...a + b]

库&导库 ​

默认 ​

dart
import 'dart:html';

使用as ​

dart
import 'package:lib1/lib1.dart';
import 'package:lib2/lib2.dart' as lib2;
// Uses Element from lib1.
Element element1 = Element();

// Uses Element from lib2.
lib2.Element element2 = lib2.Element();

仅导入库的一部分 ​

dart
// Import only foo.
import 'package:lib1/lib1.dart' show foo;

// Import all names EXCEPT foo.
import 'package:lib2/lib2.dart' hide foo;

延迟加载库 ​

  • 要延迟加载库，首先使用 导入它deferred as。
  • 当您需要该库时， loadLibrary()使用该库的标识符进行调用。 可以loadLibrary()多次调用一个库而不会出现问题。该库仅加载一次。
dart
import 'package:greetings/hello.dart' deferred as hello;
Future<void> greet() async {
  await hello.loadLibrary();
  hello.printGreeting();
}
